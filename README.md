# KinderGuard-Child-Monitoring-System
KinderGuard is an open-source Android parental control application that enables parents to monitor and manage their children's digital activity. Built as a dual-mode app (Parent + Child), it provides real-time location tracking, app usage monitoring, remote device locking, and screen time management.
# KinderGuard — Android Child Monitoring System

# ✨ Features
📍 **Real-time Location Tracking** — GPS tracking with OpenStreetMap
⚠️ **Geofencing** — Safe zone alerts
🚫 **App Blocking** — Remotely block apps
⏱️ **Screen Time Control** — Set daily usage limits
🔒 **Remote Device Lock** — Lock/unlock child's phone
🆘 **SOS Panic Button** — Emergency alerts with location
📞 **Call & SMS Monitoring** — View logs (Android 9-10)
🔗 **Secure Pairing** — Device linking via invitation codes

# 🛠️ Tech Stack
**Android Studio** (Panda 2025.3.4+)
**Java** (JDK 17)
**Firebase** (Auth, Realtime DB, Storage)
**Osmdroid** (Maps)
**Gradle** 7.5

# 📋 Requirements
Android 10 or plus
Firebase account (for backend)
2 Android devices for testing (Parent + Child)
